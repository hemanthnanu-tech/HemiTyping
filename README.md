A futuristic typing engine packed with fluid animations, hyper-smooth effects, electric UI, and dynamic challenges. Designed for people who want a fast, aesthetic, and addictively satisfying typing experience.

## ✨ Features
- **Two Unique Themes**: Switch instantly between the neon-blue `Hyper` mode and the crimson `Lust` mode with smooth curtain transitions.
- **Dynamic Challenges**: Practice with standard text (`cruise`), challenging vocabulary (`sport`), or code snippets (`code`). Or explore the flirty, dark, and seductive prompts in Lust mode!
- **Real-Time Metrics**: Track your WPM, Accuracy, Max Streak, and Raw Character count in real-time.
- **Daily Ops**: Complete daily missions to hit target WPMs in specific modes.
- **Career Stats**: Persistent career statistics saved to your local storage.
- **Immersive Audio**: Authentic web audio engine giving satisfying mechanical feedback, streak milestones, and error buzzes.

## 🚀 Quick Start

To run this project locally:

1. **Clone the repository**
```bash
git clone https://github.com/hemanthnanu-tech/HemiTyping.git
cd HemiTyping
```

2. **Install dependencies**
```bash
npm install
```

3. **Start the development server**
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser.

## 🛠️ Built With
- **React**: Core application logic.
- **Vite**: Rapid development environment.
- **Tailwind CSS**: Styling and complex animations (Blob, Electric Pulse, Glassmorphism).
- **Web Audio API**: Procedural sound generation without heavy asset files.
- **HTML Canvas**: High-performance particle confetti engine.
