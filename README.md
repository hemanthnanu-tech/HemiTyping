A futuristic typing engine packed with fluid animations, hyper-smooth effects, minimalist glassmorphism UI, and dynamic challenges. Designed for developers who want a fast, aesthetic, and addictively satisfying typing experience.

## ✨ Features
- **Prose & Code Modes**: Switch seamlessly between testing your general typing speed with dynamic prose sentences, or hone your development skills by typing out real JavaScript syntax.
- **Difficulty Scaling**: Choose between **Beginner**, **Middle**, and **Expert** datasets to push your limits.
- **Adaptive Aura Design**: The background reacts to your mode. A cool blue/emerald glow for Prose, and a deep indigo/purple glow for Code. 
- **Performance State "Speed Glow"**: When you exceed 60 WPM, the typing panel automatically pulses with a deep indigo "speed glow".
- **Real-Time Metrics**: Track your live WPM, Accuracy, and missed characters in real-time. The tracker identifies exactly what keys you are struggling with.
- **Personalized Onboarding**: Enter your name, and the text generation engine will dynamically weave your name directly into the quotes and sentences you type!
- **Career Stats & Analytics**: Persistent career statistics saved to your local storage, complete with session tracking and a line chart of your WPM history.
- **Immersive Audio & Themes**: Choose between `Mechanical` and `Thock` audio profiles for every keystroke. Instantly switch between Light and Dark mode.

## 🚀 Quick Start

To run this project locally:

1. **Clone the repository**
```bash
git clone https://github.com/hemanthnanu-tech/HemiTyping.git
cd HemiTyping
```

2. **Install dependencies**
```bash
npm install
```

3. **Start the development server**
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser.

## 🛠️ Built With
- **React**: Core application logic.
- **Vite**: Rapid development environment.
- **Tailwind CSS**: Styling and complex animations (Glassmorphism, Floating UI, Dynamic glows).
- **Web Audio API**: Procedural sound generation without heavy asset files.
- **HTML Canvas**: High-performance particle confetti engine.
